<audio src="audio.mp3" controls>
	Элемент audio не поддерживается Вашим браузером.
</audio>
тег audio используется для воспроизведеняи аудио файлов с помощью src указываем
путь к файлу а с помощью controls указываем что будет отображаться панель управления
элемент <audio> поддерживает три формата: MP3, WAV, OGG
_______________________________________________________________________________________

	<source src="audio.mp3" type="audio/mpeg">
	<source src="audio.ogg" type="audio/ogg">
</audio>

_______________________________________________________________________________________

без надобности в разрешении пользователя

<audio controls autoplay loop>
при использовании этого атрибута звуковой файл будет воспроизводится циклически

КРУГ
<svg width="1000" height="800">
	<circle cx="80" cy="80" r="50" fill="green" />
</svg>
создаем тег svg для рисования задаем размеры его 
и рисуем на нём круг 
circle - круг
cx - центр круга передвегается вправо по экрану
cy - центр круга передвегается вниз по экрану
r - радиус
fill - задаёт цвет нашему кругу
stroke - для создания контура
_______________________________________________________________________________________
ПРЯМОУГОЛЬНИК

	x="20" y="20" fill="green"/>

ЛИНИЯ
<svg width="400" height="410">

	<line x1="10" y1="10" x2="200" y2="100" 
	style="stroke:#000000; stroke-linecap:round; 
	stroke-width:20" />
</svg>
line - линия
x1 - первая координата икс с которой берёт начало линия
y1 - первая координата игрик с которой берёт начало линия
x2 - вторая координата икс которой заканчивается линия
y2 - вторая координата игрик которой заканчивается линия
_______________________________________________________________________________________

<svg width="1000" height="800">
	<rect width="50" height="50" fill="orange">
		<animate attributeName="x" from="0" 
		to="300" dur="3s" fill="freeze" repeatCount="2" />
	</rect>
</svg>
attributeName - указывает к какому атрибуту будет применятся анимация то есть к 
координате иксу
from - назначает стартовое значение атрибута 
to - назначает конечное значение атрибута
dur - назначает длительность анимации
fill - указывает должно ли значение атрибута вернутся к своему первоначальному значению 
после завершения анимации (значение remove  - сбрасывает значение\ freeze -  сохраняет
значение to)
repeatCount - назначает количество повторов анимации(если указать занчение indefinite
то анимация будет бесконечно повторятся)
_______________________________________________________________________________________
M: moveto
L: lineto
H: horizontal lineto
V: vertical lineto
C: curveto
S: smooth curveto
Q: quadratic Bezier curve
T: smooth quadratic Bezier curveto
A: elliptical Arc
Z: closepath

<svg width="1000" height="800">
	<path d="M 0 0 L200 200 L200 0 Z"
	style="stroke:#000; fill:none;" />
</svg>
M станавливает нашу виртуальную ручку в положении (0,0). Ручка затем двигается по 
диагонали вниз и вправо на 200 пикселей затем вверх в положение (200,0). Команда Z
замыкает фигуру, создается гипотенуза

_______________________________________________________________________________________
<form autocomplete="off"> 
	<label for="email">Ваш электронный адрес
	</label>
	<input type="text" name="email" placeholder="email@example.com" autofocus 
	required />
</form>
for="email" - привязываем метку к данному полю для ввода текста 
placeholder - подсказка для пользователя какая информация может быть введена в поле
autofocus - ставит курсор в нужное поле вводакогда форма загрузится
required - указывает пользователю что хаполнение формы необходимо
autocomplete - указываем будет ли автозаполняться ранее введёные данные пользователем 
в поля для ввода
